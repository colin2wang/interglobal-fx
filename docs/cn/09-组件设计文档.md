## 2. React 组件详细说明

### 2.1 CandlestickChart（K线图表）

**用途**: 展示外汇K线图表，支持多周期切换
**路径**: components/chart/CandlestickChart.tsx

**Props：**

| 属性名 | 类型 | 必填 | 默认值 | 说明 |
|--------|------|------|--------|------|
| symbol | string | 是 | - | 品种代码 |
| period | string | 否 | '1h' | K线周期：1m/5m/15m/1h/4h/1d |
| height | number | 否 | 400 | 图表高度 |
| showVolume | boolean | 否 | true | 是否显示成交量 |
| onPeriodChange | (period: string) => void | 否 | - | 周期切换回调 |

**Events：**

| 事件名 | 参数 | 说明 |
|--------|------|------|
| onCandleClick | (candle: CandleData) => void | 点击K线 |
| onCrosshairMove | (data: CrosshairData) => void | 十字光标移动 |

**Slots：**

| 插槽名 | 说明 |
|--------|------|
| toolbar | 工具栏区域 |
| legend | 图例区域 |

**使用示例：**
```tsx
import { CandlestickChart } from '@/components/chart';

<CandlestickChart
  symbol="EURUSD"
  period="1h"
  height={500}
  showVolume={true}
  onPeriodChange={(p) => setPeriod(p)}
  onCandleClick={(candle) => console.log(candle)}
/>
```

## 5. 组件开发规范

### 5.1 React 组件规范

- 组件使用函数组件 + Hooks
- Props 接口使用 TypeScript 定义
- 组件放在 components 目录，按功能模块分类
- 公共组件提供完整的 JSDoc 注释
- 样式使用 CSS Modules 或 styled-components

### 5.2 Vue 组件规范

- 使用 Composition API（setup语法糖）
- Props 使用 defineProps 定义
- 组件放在 components/common 或 components/form 等目录
- 公共组件支持 v-model 双向绑定
- 样式使用 SCSS，使用 BEM 命名

### 5.3 Flutter 组件规范

- 使用 StatelessWidget 或 StatefulWidget
- 使用 const 构造不可变组件
- 组件放在 widgets 目录
- 使用 Provider 或 BLoC 进行状态管理
- 遵循 Material Design 设计规范
