# 环球汇通外汇交易平台 - 数据库详细设计

## 1. ER 关系描述

### 1.1 核心实体关系

```
租户(cfg_tenant) 1---N 账户(acc_account) 1---N 钱包(wlt_wallet)
账户(acc_account) 1---N 订单(trd_order) 1---N 持仓(pos_position)
持仓(pos_position) N---1 货币对(sym_symbol)
订单(trd_order) N---1 货币对(sym_symbol)
货币对(sym_symbol) N---1 报价(qte_tick)
账户(acc_account) 1---N 客户(crm_customer) 1---N KYC(kyc_application)
客户(crm_customer) N---1 IB合伙人(ib_partner)
IB合伙人(ib_partner) 1---N IB合伙人(ib_partner) (自关联层级)
客户(crm_customer) 1---N 工单(tic_ticket) 1---N 工单回复(tic_reply)
客户(crm_customer) 1---N 跟进记录(fll_follow)
账户(acc_account) 1---N 入金记录(txn_deposit)
账户(acc_account) 1---N 出金记录(txn_withdrawal)
持仓(pos_position) 1---1 日结清(clr_daily_settlement)
账户(acc_account) 1---N 风控事件(rsk_event)
风控规则(rsk_rule) N---1 货币对(sym_symbol)
```

## 2. 数据表详细设计

### 2.1 货币对表（sym_symbol）

**表说明：** 定义所有可交易的外汇货币对、加密货币、贵金属、指数等品种

| 字段名 | 类型 | 长度 | 必填 | 默认值 | 说明 |
|--------|------|------|------|--------|------|
| id | BIGSERIAL | - | 是 | 自增 | 主键 |
| symbol | VARCHAR | 20 | 是 | - | 货币对代码，如EURUSD |
| symbol_name | VARCHAR | 100 | 是 | - | 货币对名称，如欧元/美元 |
| symbol_type | SMALLINT | - | 是 | - | 品种类型：1-主要货币对 2-次要货币对 3-exotic 4-贵金属 5-指数 6-加密货币 |
| base_currency | VARCHAR | 10 | 是 | - | 基准货币 |
| quote_currency | VARCHAR | 10 | 是 | - | 报价货币 |
| precision | SMALLINT | - | 是 | 5 | 价格精度（小数位数） |
| lot_size | DECIMAL | 15,5 | 是 | 100000 | 标准手大小 |
| min_lot | DECIMAL | 15,5 | 是 | 0.01 | 最小交易量（手） |
| max_lot | DECIMAL | 15,5 | 是 | 100 | 最大交易量（手） |
| lot_step | DECIMAL | 15,5 | 是 | 0.01 | 交易量步进 |
| min_distance_points | INTEGER | - | 是 | 10 | 止盈止损最小距离（点） |
| trade_mode | SMALLINT | - | 是 | 1 | 交易模式：1-可交易 2-仅查看 3-停用 |
| contract_size | DECIMAL | 15,5 | 是 | 100000 | 合约规模 |
| swap_long | DECIMAL | 10,5 | 是 | - | 多头隔夜利息(点) |
| swap_short | DECIMAL | 10,5 | 是 | - | 空头隔夜利息(点) |
| swap_mode | SMALLINT | - | 是 | 1 | 利息计算模式：1-点 2-百分比 |
| trading_hours | VARCHAR | 100 | 是 | - | 交易时间配置 |
| status | SMALLINT | - | 是 | 1 | 状态：1-启用 2-禁用 |
| tenant_id | BIGINT | - | 是 | - | 租户ID |
| created_by | BIGINT | - | 是 | - | 创建人ID |
| created_time | TIMESTAMPTZ | - | 是 | NOW() | 创建时间 |
| updated_by | BIGINT | - | 否 | NULL | 修改人ID |
| updated_time | TIMESTAMPTZ | - | 否 | NULL | 修改时间 |
| is_deleted | SMALLINT | - | 是 | 0 | 逻辑删除 |
| remark | VARCHAR | 500 | 否 | NULL | 备注 |

**索引设计：**

| 索引名 | 类型 | 字段 | 说明 |
|--------|------|------|------|
| pk_sym_symbol | 主键 | id | 主键索引 |
| uk_sym_symbol_code | 唯一 | symbol, tenant_id | 货币对代码唯一 |
| idx_sym_type | 普通 | symbol_type | 按品种类型查询 |

**DDL 脚本：**
```sql
CREATE TABLE sym_symbol (
    id BIGSERIAL PRIMARY KEY,
    symbol VARCHAR(20) NOT NULL,
    symbol_name VARCHAR(100) NOT NULL,
    symbol_type SMALLINT NOT NULL DEFAULT 1,
    base_currency VARCHAR(10) NOT NULL,
    quote_currency VARCHAR(10) NOT NULL,
    precision SMALLINT NOT NULL DEFAULT 5,
    lot_size DECIMAL(15,5) NOT NULL DEFAULT 100000,
    min_lot DECIMAL(15,5) NOT NULL DEFAULT 0.01,
    max_lot DECIMAL(15,5) NOT NULL DEFAULT 100,
    lot_step DECIMAL(15,5) NOT NULL DEFAULT 0.01,
    min_distance_points INTEGER NOT NULL DEFAULT 10,
    trade_mode SMALLINT NOT NULL DEFAULT 1,
    contract_size DECIMAL(15,5) NOT NULL DEFAULT 100000,
    swap_long DECIMAL(10,5) NOT NULL DEFAULT 0,
    swap_short DECIMAL(10,5) NOT NULL DEFAULT 0,
    swap_mode SMALLINT NOT NULL DEFAULT 1,
    trading_hours VARCHAR(100) NOT NULL DEFAULT '00:00-24:00',
    status SMALLINT NOT NULL DEFAULT 1,
    tenant_id BIGINT NOT NULL,
    created_by BIGINT NOT NULL,
    created_time TIMESTAMPTZ NOT NULL DEFAULT NOW(),
    updated_by BIGINT DEFAULT NULL,
    updated_time TIMESTAMPTZ DEFAULT NULL,
    is_deleted SMALLINT NOT NULL DEFAULT 0,
    remark VARCHAR(500) DEFAULT NULL,
    UNIQUE(symbol, tenant_id)
);

CREATE INDEX idx_sym_type ON sym_symbol(symbol_type);
```

| order_no | UUID | - | 是 | - | 订单号（UUID） |
| tenant_id | BIGINT | - | 是 | - | 租户ID |
| account_id | BIGINT | - | 是 | - | 账户ID |
| symbol_id | BIGINT | - | 是 | - | 货币对ID |
| order_type | SMALLINT | - | 是 | - | 订单类型：1-市价单 2-限价单 3-止损单 4-止盈单 5-止损限价单 6-挂单 |
| order_side | SMALLINT | - | 是 | - | 订单方向：1-买入 2-卖出 |
| order_price | DECIMAL | 20,10 | 否 | NULL | 订单价格（限价单用） |
| stop_price | DECIMAL | 20,10 | 否 | NULL | 触发价格（止损/挂单用） |
| lot_size | DECIMAL | 15,5 | 是 | - | 订单数量（手） |
| filled_lot | DECIMAL | 15,5 | 是 | 0 | 已成交数量 |
| avg_price | DECIMAL | 20,10 | 否 | NULL | 成交均价 |
| order_price_show | DECIMAL | 20,10 | 否 | NULL | 用户下单时的显示价格 |
| slippage_points | INTEGER | - | 是 | 0 | 滑点（点） |
| slippage_type | SMALLINT | - | 是 | 1 | 滑点类型：1-固定 2-百分比 |
| status | SMALLINT | - | 是 | 1 | 状态：1-挂单中 2-部分成交 3-完全成交 4-已取消 5-已拒绝 6-过期 |
| execution_type | SMALLINT | - | 是 | 1 | 执行类型：1-全单执行 2-逐笔执行 |
| time_in_force | SMALLINT | - | 是 | 1 | 有效期：1-GTC永久 2-GFD当日有效 3-GTD指定日期 |

| ip_address | INET | - | 否 | NULL | 下单IP |
| order_text | VARCHAR | 500 | 否 | NULL | 订单备注 |
| reason | VARCHAR | 500 | 否 | NULL | 拒绝/取消原因 |
| risk_check_status | SMALLINT | - | 是 | 1 | 风控审核状态：1-通过 2-待审核 3-拒绝 |
| risk_check_time | TIMESTAMPTZ | - | 否 | NULL | 风控审核时间 |
| risk_rule_ids | VARCHAR | 200 | 否 | NULL | 触发风控规则ID列表 |
| source_order_no | UUID | - | 否 | NULL | 来源订单号（反手/修改用） |

| updated_by | BIGINT | - | 否 | NULL | 修改人ID |
| updated_time | TIMESTAMPTZ | - | 否 | NULL | 修改时间 |
| is_deleted | SMALLINT | - | 是 | 0 | 逻辑删除 |
| remark | VARCHAR | 500 | 否 | NULL | 备注 |

| 索引名 | 类型 | 字段 | 说明 |
|--------|------|------|------|
| pk_trd_order | 主键 | id | 主键索引 |
| uk_trd_order_no | 唯一 | order_no | 订单号唯一 |

    channel SMALLINT NOT NULL DEFAULT 1,
    ip_address INET DEFAULT NULL,
    order_text VARCHAR(500) DEFAULT NULL,
    reason VARCHAR(500) DEFAULT NULL,
    risk_check_status SMALLINT NOT NULL DEFAULT 1,
    risk_check_time TIMESTAMPTZ DEFAULT NULL,
    risk_rule_ids VARCHAR(200) DEFAULT NULL,
    source_order_no UUID DEFAULT NULL,
    created_by BIGINT NOT NULL,
    created_time TIMESTAMPTZ NOT NULL DEFAULT NOW(),
    updated_by BIGINT DEFAULT NULL,
    updated_time TIMESTAMPTZ DEFAULT NULL,
    is_deleted SMALLINT NOT NULL DEFAULT 0,
    remark VARCHAR(500) DEFAULT NULL,
    UNIQUE(order_no)

CREATE INDEX idx_trd_account ON trd_order(account_id);
CREATE INDEX idx_trd_symbol ON trd_order(symbol_id);
CREATE INDEX idx_trd_status ON trd_order(status);
CREATE INDEX idx_trd_created ON trd_order(created_time);

## 5. DDL 脚本规范

- 所有表名使用小写+下划线，带模块前缀
- 字段注释（COMMENT）必须填写，说明字段业务含义
- 枚举类型字段使用 SMALLINT，不使用 ENUM 类型
- 金额字段使用 DECIMAL(20,8)，禁止使用 FLOAT/DOUBLE
- 价格字段使用 DECIMAL(20,10)，精度更高
- 时间字段使用 TIMESTAMPTZ（UTC时区）
- 所有外键关系通过应用层维护，数据库层不建外键约束
- 索引命名：普通索引 idx_，唯一索引 uk_，联合索引 idx_（多字段下划线连接）
- 重要业务表使用乐观锁（version字段）防止并发问题
- TimescaleDB超表需要单独创建
