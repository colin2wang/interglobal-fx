## 2. 交易终端页面说明

### 2.1 交易主页面

**路由**: /trading
**组件**: pages/trading/main.tsx
**权限**: 登录用户

**页面布局：**
```
+--------------------------------------------------------------------------+
|  [Logo]  环球汇通    [EURUSD][GBPUSD][USDJPY][...]   [账户] [消息] [设置] |
+--------------------------------------------------------------------------+
|                    |                              |                      |
|   货币对列表        |       K线图表区域             |    交易面板          |
|  +--------------+  |  +------------------------+  |  +----------------+  |
|  | EURUSD  1.08 |  |  |                        |  |  |  Symbol       |  |
|  | GBPUSD  1.27 |  |  |    [K线图表]           |  |  |  EURUSD       |  |
|  | USDJPY 149.5 |  |  |                        |  |  +----------------+  |
|  | ...         |  |  |                        |  |  | [买入] [卖出]  |  |
|  +--------------+  |  +------------------------+  |  |  手数: [0.10]  |  |
|                    |  [1M][5M][15M][1H][4H][1D] |  |  市价单       |  |
|   市场资讯         |                              |  +----------------+  |
|  +--------------+  |  市场深度                    |                      |
|  | 新闻标题...  |  |  卖盘: 1.08530 (100)        |    快捷订单          |
|  | 财经日历... |  |  卖盘: 1.08525 (250)        |  +----------------+  |
|  +--------------+  |  --------------------------  |  | Buy 0.10 EURUSD|  |
|                    |  买盘: 1.08520 (150)        |  | Sell 0.10 EURUSD| |
|                    |  买盘: 1.08515 (300)        |  +----------------+  |
+--------------------------------------------------------------------------+
```

**功能说明：**
1. 左侧货币对列表：点击切换交易品种
2. 中间K线图表：支持多周期切换、手动缩放、十字光标
3. 右侧交易面板：下单表单、快捷交易按钮
4. 实时报价：WebSocket推送，毫秒级更新

```
+----------------------------------------------------------------------+
|  [分页]  共 1,234 条  < 1 2 3 ... 10 >  每页 [20▼] 条              |

```

**搜索条件：**

| 字段 | 组件类型 | 字典/选项 | 默认值 |
|------|----------|-----------|--------|
| 订单号 | 输入框 | - | - |
| 品种 | 下拉选择 | symbol_dict | 全部 |
| 状态 | 下拉选择 | order_status | 全部 |
| 账户 | 输入框 | - | - |
| 类型 | 下拉选择 | order_type | 全部 |
| 时间范围 | 日期范围 | - | 最近7天 |

**表格列定义：**

| 列名 | 字段 | 宽度 | 对齐 | 格式化 | 排序 |
|------|------|------|------|--------|------|
| 序号 | index | 60px | center | - | 否 |
| 订单号 | orderNo | 180px | left | 复制按钮 | 否 |
| 账户 | accountNo | 120px | left | 跳转详情 | 否 |
| 品种 | symbol | 100px | center | 货币对 | 否 |
| 类型 | orderType | 80px | center | 字典 | 否 |
| 方向 | orderSide | 60px | center | 买入(红)/卖出(绿) | 否 |
| 数量 | lotSize | 80px | right | 小数点后2位 | 是 |
| 价格 | price | 100px | right | 精度跟随品种 | 是 |
| 状态 | status | 80px | center | 标签色 | 否 |
| 时间 | createdTime | 160px | center | yyyy-MM-dd HH:mm:ss | 是 |
| 操作 | action | 120px | center | 按钮组 | 否 |

**字典数据映射：**

| 字段 | 字典类型 | 显示方式 |
|------|----------|----------|
| orderType | order_type | 标签 |
| orderSide | order_side | 买入(红色)/卖出(绿色) |
| status | order_status | 标签，不同状态不同颜色 |

**操作按钮：**

| 按钮 | 权限标识 | 交互 | 说明 |
|------|----------|------|------|
| 详情 | order:view | 抽屉/弹窗 | 显示订单完整信息 |
| 取消 | order:cancel | 确认弹窗 | 取消未成交订单 |
| 重试 | order:manage | 直接执行 | 重试失败订单 |

**交互规则：**
1. 页面加载时自动查询列表
2. 搜索条件变化后点击查询，重置为第1页
3. 点击表格行可展开查看订单详情
4. 订单状态实时更新（WebSocket推送）
5. 导出支持Excel格式

### 5.2 KYC认证页面

**路由**: /kyc (trader)
**组件**: pages/profile/kyc.tsx

**认证步骤：**
1. 个人信息填写
2. 证件上传（身份证正反面）
3. 人脸识别验证
4. 地址证明上传
5. 提交审核
