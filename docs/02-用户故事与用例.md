# 环球汇通外汇交易平台 - 用户故事与用例文档

## 1. 概述

本文档以"角色-操作-预期结果-异常分支"格式描述环球汇通外汇交易平台所有用户故事，确保可直接用于接口和页面逻辑生成。

## 2. 用户故事列表

### US-001: 零售交易者下单交易

- **角色**: 零售交易者
- **故事**: 作为零售交易者，我希望快速下单交易，以便抓住市场机会获利
- **优先级**: P0
- **所属模块**: M02-交易订单系统

**正常流程：**
1. 用户登录交易终端
2. 用户选择EURUSD货币对
3. 用户设置交易量0.1手
4. 用户选择市价单，点击买入
5. 系统校验保证金充足
6. 系统执行订单
7. 系统显示订单成交，持仓显示开仓成功

**预期结果：**
- 订单在2秒内成交
- 持仓列表显示EURUSD多头0.1手
- 账户余额扣除所需保证金
- 成交记录显示成交价格和时间

**异常分支：**

| 异常场景 | 触发条件 | 系统处理 | 用户提示 |
|----------|----------|----------|----------|
| 保证金不足 | 可用保证金 < 所需保证金 | 拒绝下单 | "保证金不足，当前可用：XXX" |
| 品种停牌 | 交易时间外或品种已停用 | 拒绝下单 | "该货币对当前不可交易" |
| 价格大幅波动 | 报价与用户查看价格差异>1% | 提示确认 | "价格已变动，是否接受新价格？" |
| 网络超时 | 订单请求超时 | 重试3次后失败 | "网络异常，请重试" |

**验收标准：**
- [ ] 订单从提交到成交不超过2秒
- [ ] 成交价格与提交时价格偏差不超过滑点设置
- [ ] 持仓和余额实时更新

### US-002: 设置止盈止损

- **角色**: 零售交易者
- **故事**: 作为零售交易者，我希望设置止盈止损，以便控制风险锁定利润
- **优先级**: P0
- **所属模块**: M02-交易订单系统

**正常流程：**
1. 用户查看持仓列表
2. 用户选择持仓，点击"设置止盈止损"
3. 用户设置止盈价1.1200，止损价1.1000
4. 系统保存止盈止损订单
5. 当价格触及1.1200时，系统自动平仓
6. 系统推送成交通知

**预期结果：**
- 止盈止损订单状态显示"挂单中"
- 价格触及条件后自动成交
- 成交记录标注为"止盈平仓"

**异常分支：**

| 异常场景 | 触发条件 | 系统处理 | 用户提示 |
|----------|----------|----------|----------|
| 止盈止损间距过小 | 止盈止损距离 < 品种最小距离 | 拒绝设置 | "止盈止损间距不能小于X点" |
| 止损超过限制 | 止损设置不符合监管要求 | 拒绝设置 | "止损距离不符合监管要求" |

**验收标准：**
- [ ] 止盈止损可随时修改或删除
