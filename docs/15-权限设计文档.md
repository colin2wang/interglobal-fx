# 环球汇通外汇交易平台 - 权限设计文档

## 1. 权限模型

采用 RBAC（基于角色的访问控制）模型：

```
用户(User) ---N:N--- 角色(Role) ---N:N--- 权限(Permission)
                    |
                    v
              数据权限(DataScope)
```

- 用户可分配多个角色
- 角色可关联多个权限
- 权限分为：菜单权限、按钮权限、接口权限
- 支持数据权限控制（按租户/部门/个人）

## 5. 数据权限

| 数据范围 | 说明 | SQL条件 |
|----------|------|----------|
| 1-全部数据 | 可查看所有数据 | 无附加条件 |
| 2-本租户数据 | 仅查看本租户数据 | tenant_id = {当前租户ID} |
| 3-本部门数据 | 仅查看本部门数据 | dept_id = {当前部门ID} |
| 4-仅本人数据 | 仅查看自己创建的数据 | created_by = {当前用户ID} |

## 8. 权限验证流程

```
请求 → 网关 → 解析Token → 获取用户ID → 查询用户角色 → 查询角色权限 → 校验权限 → 放行/拒绝
```
