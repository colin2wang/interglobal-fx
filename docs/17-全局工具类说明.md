### 2.1 SecurityUtils - 安全工具

**用途**: 用户认证、密码加密、Token解析等安全相关操作
**路径**: common/util/SecurityUtils.java

**方法列表：**

| 方法名 | 参数 | 返回值 | 说明 |
|--------|------|--------|------|
| getCurrentUserId | () | Long | 获取当前登录用户ID |
| getCurrentUsername | () | String | 获取当前登录用户名 |
| getCurrentTenantId | () | Long | 获取当前租户ID |
| getCurrentAccountId | () | Long | 获取当前交易账户ID |
| encryptPassword | String password | String | BCrypt加密密码 |
| matchesPassword | String rawPassword, String encodedPassword | boolean | 密码校验 |
| generateToken | Long userId, String username | String | 生成JWT Token |
| parseToken | String token | Claims | 解析Token |
| getUserIdFromToken | String token | Long | 从Token获取用户ID |
| isTokenExpired | String token | boolean | 判断Token是否过期 |
| getPermissions | () | Set<String> | 获取当前用户权限集合 |
| hasPermission | String permission | boolean | 校验是否有某权限 |
| hasAnyPermission | String... permissions | boolean | 校验是否有任一权限 |

**使用示例：**

```java
// 获取当前用户ID
Long userId = SecurityUtils.getCurrentUserId();

// 密码加密
String encodedPassword = SecurityUtils.encryptPassword("Password123");

// 密码校验
boolean matches = SecurityUtils.matchesPassword("Password123", encodedPassword);

// 权限校验
if (SecurityUtils.hasPermission("order:create")) {
    // 有下单权限
}

// 生成Token
String token = SecurityUtils.generateToken(userId, username);
```

```go
    logger.String("symbol", symbol),
    logger.Float64("lot_size", lotSize),
)
```
